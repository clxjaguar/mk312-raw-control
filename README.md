# mk312-raw-control

This computer program is sending the same data than the master box when two mk312 are linked together. Fetch your oscilloscope, and have fun!

![screenshot](https://github.com/user-attachments/assets/22c32ab0-7a7f-4960-9ecb-d55a1d931059)

This program was made after sniffing the communication between two real mk312 boxes:

[![Linking two MK312 boxes together with a cross cable and decoding data in real time](https://img.youtube.com/vi/WbT4LLAQTqE/0.jpg)](https://www.youtube.com/watch?v=WbT4LLAQTqE)
